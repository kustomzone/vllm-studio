Features

🚀 One-Click Launch

Launch any model from your library with a single click. Automatic GPU memory management and optimized settings.

📝 Recipe System

Save and reuse model configurations as recipes. Share your optimal settings with the community.

💬 Built-in Chat

Test your models instantly with the integrated chat interface. Supports streaming and tool calls.

📊 Real-time Monitoring

Live GPU utilization, memory usage, and inference metrics. Prometheus + Grafana dashboards included.

🔌 OpenAI Compatible

LiteLLM integration for OpenAI API compatibility. Drop-in replacement for your existing applications.

🔧 vLLM & SGLang

Supports both vLLM and SGLang backends. Switch between them based on your needs.

Quick Start

# Install the controller pip install vllm-studio # Run the controller (starts API on :8080) vllm-studio # (Optional) Run the web UI cd frontend && npm install && npm run dev # Or use Docker docker-compose up -d
